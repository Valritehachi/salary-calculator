# Salary Calculator

A comprehensive web application for calculating various salary-related metrics, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Salary to Hourly Converter**: Convert annual salary to hourly, daily, weekly, and monthly rates, accounting for PTO.
- **After-Tax Calculator**: Estimate take-home pay after federal and state taxes.
- **Cost of Living Comparison**: Compare salaries across different cities.
- **Job Offer Comparison**: Compare multiple job offers side-by-side.
- **Raise Calculator**: Calculate salary increases and extra earnings.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To build for production:
```bash
npm run build
```

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
